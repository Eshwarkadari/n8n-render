# n8n on Render

n8n workflow automation setup deployed on Render for free hosting.

## What is n8n?
n8n is a free and open-source workflow automation tool — like Zapier but self-hosted.

## Features
- Self-hosted n8n instance
- Deployed on Render
- Automate any workflow
